1. Modely a vztahy
Vytvoříme následující modely:

Kniha (Book) - může mít více autorů
Autor (Author) - může napsat více knih
Kategorie (Category) - kniha může patřit do více kategorií
Hodnocení (Rating) - polymorfní vztah (může se vztahovat ke knihám i autorům)
Komentář (Comment) - polymorfní vztah (může se vztahovat ke knihám i autorům)

PS C:\docker\taktik-api-knihy> php artisan make:model Book
PS C:\docker\taktik-api-knihy> php artisan make:model Author
PS C:\docker\taktik-api-knihy> php artisan make:model Category
PS C:\docker\taktik-api-knihy> php artisan make:model Rating
PS C:\docker\taktik-api-knihy> php artisan make:model Comment

2 Migrace
PS C:\docker\taktik-api-knihy> php artisan make:migration books
PS C:\docker\taktik-api-knihy> php artisan make:migration authors
PS C:\docker\taktik-api-knihy> php artisan make:migration categories
PS C:\docker\taktik-api-knihy> php artisan make:migration ratings
PS C:\docker\taktik-api-knihy> php artisan make:migration comments
PS C:\docker\taktik-api-knihy> php artisan make:migration book_category

3 API Kontrolery
php artisan make:controller Api/BookController --api
php artisan make:controller Api/AuthorController --api
php artisan make:controller Api/CategoryController --api

4 API Resources
php artisan make:resource BookResource
php artisan make:resource AuthorResource
php artisan make:resource CategoryResource
php artisan make:resource RatingResource
php artisan make:resource CommentResource

5 Reqest - Validační požadavky
php artisan make:request BookRequest
php artisan make:request AuthorRequest
php artisan make:request CategoryRequest


php artisan make:factory RatingFactory --model=Rating
